Workspace

Control

This page is the workspace. In this page, you can use the hotkeys to add labels or check result. With the default hotkeys, you can add grade-A, B, C or non-lens labels by pressing the "1", "2", "3" or "0" keys. The "j" and "k" keys help you get the next or previous image.
Note: When you go to the last image, you cannot get the next image by pressing the “j” key.

Filter

There are some filters for rolling back to the first image of some specific groups: all images, grade-A, B, C, non-lens and unlabeled images. When you open the workspace page for the first time, the default filter is "All images". In some cases, the "unlabeled" filter is a good choice for getting all unlabeled images.

If the filter gets no image, it will return all images whatever they are labeled or not.
